HEML is an open source markup language for building responsive email.

It takes care of some email formatting troubleshooting.

# Usage

Use online editor on https://heml.io/editor or install locally via NPM:

```
npm install heml -g

heml develop email.heml  // live develop

heml build email.heml  // build HTML
```
